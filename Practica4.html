Tipos de datos en MySQL

Varchar

Se usa para almacenar cadenas de caracteres. Una cadena es una secuencia de caracteres. Se coloca entre comillas(simples): 'Hola',
El tipo 'Varchar' define una cadena de longitud variable en la cual determinamos el maximo de caracteres. Puede guardar hasta 255 caracteres. Para almacenar cadena de hasta 30 caracteres, definimos un campo de tipo varchar(30).

int

se usa para guardar valores numeros enteros. de - 2000000 a 2000000000 aproximadamente.
Definimos campos de este tipo cuando queremos representar. Por ejemplo, cantidades.
