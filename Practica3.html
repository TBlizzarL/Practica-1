SQL, Structure Query, Language(Lenguaje de consulta Estructurando) es un lenguaje de programacion para trabajar con base de datos relacionales como MySQL, Oracle, etc.
MySQL es un intrepretador de SQL, es un servidor de base de datos.
MySQL permite crear base de datos y tablas, insertar datos, modiciarlos, eliminarlos, ordenarlos, hacer consultas y realizar muchas operaciones, etc, resumiendo; administra base datos.

Este tutorial tiene como objetivo acercar los conceptos iniciales para introducirse en el mundo de las bases de datos.
